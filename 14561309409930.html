跨平台广告传播

2016/2/22

课程概要

本课程将分析品牌及传播环境及受众需求以订定合适的跨平台广告计划， 并通过了解受众需求及
习惯选择最有影响力的广告讯息，根据不同平台特性, 与广告代理商合作计划及执行整合广告策略
并评估跨平台广告计划的成效。

课程目标

顺利完成本课程后，学员将能够：
1. 严谨分析品牌及传播环境及受众需求以订定合适的跨平台广告计划；
2. 通过了解受众需求及习惯选择最有影响力的广告讯息；
3. 根据不同平台特性, 与广告代理商合作计划及执行整合广告策略；
4. 批判性评估跨平台广告计划的成效。

学习资料

必读教材:

贝尔齐等着, 郑苏晖等译 (2009), 广告与促销：整合营销传播视角（第八版）,
中国人民大学出版社

推荐读物:

营销管理，作者：菲利浦・科特勒 / 凯文・莱恩・凯勒
译者: 王永贵 / 于洪彦 / 何佳讯 / 陈荣，上海：格致出版社：上海人民出版
社
定位（Positioning），作者：阿尔・里斯 / 杰克・特劳特
译者: 谢伟山 / 苑爱冬
营销革命 3.0，作者：菲利普・科特勒, 译者: 毕崇毅, 机械工业出版社
