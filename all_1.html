新产品平台化

多平台和多名称属于一个产品内的问题，在单产品内建立能力并做到组织能力可分裂，在新产品启动的时候能够分裂出相关的组织在内部实施，感觉是比较靠谱的做法。如果多平台和多名称并列于多产品，管理难度会几何级增加。

所以，新产品部的产品大战略定位于：多产品平台化。从周二的讨论会上来看，战略先行对于战略达成的重要性影响很大，我也会从现在开始，主要着手思考和执行战略。

多产品的大逻辑：

在多产品的演进方向上，目前整理几个点：

  1. 用户内在需求（婚恋、约会等）
  2. 基础体验上平行拓展 （短视频、音频、直播），对于平行拓展，只要强化女性用户主动交互的核心点（基本判断是通过自然交友很难引发大众用户的转化）
  3. 用户群体 （低端、中端、高端）
  4. 社交演进趋势上的大杀器 （直播类等）

从现在我着手盯以下几个事情：

  1. 产品跨代升级的成功率 （有严欣、程远炜、王楠 坐镇，问题不大）
  2. 拍拖多平台的构建 （微信需要一线经理，同占潮已经沟通数次。 微信新产品已经跑偏，是否需要在公司层面干预）
  3. 拍拖产品模块组织的人员梯队建设
  4. 新产品逻辑的思考及后备产品leader的培养或获取

战略讨论

2016/7/31

时值年度过半，我想在公司高层层面召开针对公司的中长期战略的讨论会。具体讨论内容可以围绕以下几个问题：

1、公司上半年的发展状况怎么样（不用数字，可以凭感觉定性描述）？

  1. 从营收上来看：春节高点不够高；节后衰退明显；整体走势弱于去年。结构风险大，后劲乏力！

  2. 从业务上开看：上半年产品没有跨越发展；营利能力没有明显提升；渠道上也没有类似去年微信的突破；各业务单元通过管理优化吃到了我们在现有体量下应吃下的份额。

  3. 管理上来看，目前的管理结构比较适合将当前业务效率最大化，形成了几个单一任务的团队（士正，江隆，封涛，广志，李岩，开山），较为复杂的任务执行起来困难，但已经开始逐步意识到为发展做准备。

  4. 明确了防守任务和进攻任务

2、这种状况是哪些因素导致的（外部、内部）？

外部：
无线红利消失；渠道集中化、成本上升；珍爱等竞品加入微信竞争；
核心本质还是继续着体量优势，无法持续在市场的竞争中保持领先。

内部：
各部门各自为小目标负责，而目标之间的联动无人协调。

新产品的孵化机制正在摸索；产品落后市场迭代趋势

品牌正向化和营利能力、吸量能力的冲突；
对“有缘网”传统婚恋产品的目标定位不确定；
产品运营资源耗散在内部优化的小项目上；
管理结构调整

3、在这个过程中，我们做错了哪些事情？为什么会做错？未来还有没有可能犯类似的错？

过度相信经验，以经验代替市场，盲目乐观，低估实际困难（新产品的『填油』投入）
没有摆正防守任务和进攻任务的位置（在“有缘网”品牌任务上投入过多）
多线并进；决心不足；

4、在这个过程中，我们又做对了哪些事？我们为什么能做对？未来能不能继续对？

  1. 战略上统一了继续发展的思想，理出进攻和防守的关键点，集中资源，All in （新产品研发换代任务，渠道能力的建设）;
  2. 管理结构的优化， 引入汉文、严欣 新能力的加入；
  3. 日常运营向渠道侧倾斜，以渠道需求驱动老产品常规运营；
  4. 有缘网任务定位，及时收缩品牌投入；

5、如果我们希望在2018年婚恋业务收入能达到全年15亿元（40%的年增长），我们必须做好哪几件最重要的事？为什么？

任务分解：
有缘网：    1亿  继续强化婚恋领先2B品牌，塑造行业地位。
多名称：   4亿 （交友）  老交友产品丢给代理商
新产品：   10亿（arpu  提升 50%，一天至少 40万新增） 渠道杠杆能2-3倍于arpu提升
  1. 婚恋收入指的是婚恋+交友的总收入
  2. 夯实新产品的跨代优势，以此为基础展开多产品/多名称矩阵，完全覆盖婚恋交友需求
    1. 产品对用户的可玩性，符合产品迭代的趋势 （吸量）
    2. 继承深化营收的能力
    3. 形成正向口碑的生态
  3. 渠道必须形成并行推动多产品运营，交叉覆盖获取流量的能力，平衡流量扩张和产品营收波动的能力（建立单一任务的单一团队）
  4. 市场必须形成以公关为主的品推能力，强化有缘网婚恋领先品牌，并帮助新品牌快速上位，口碑爆发式增长

安全性
{公司的安全性和产品的安全性}

6、要达到18年的目标，我们最主要的几个风险是什么？我们应该从哪几个方向控制这几个风险呢？

  1. 新产品产品力和营收能力形不成压倒性优势。 应对：快速验证；及时调整；预先设计备份方案
  2. 渠道情况急剧变化，微信的合作发生战略变化。 应对：加强市场环境监测；加深和微信的运营层连接
  3. 黑天鹅，致命负面新闻。   应对：严格控制正向化底线；对高风险的策略和支付问题零容忍；提前公关布局

附加思考（选答，但建议大家思考一下）：

7、如果我们希望在2020年，达到100亿的年收入和30亿以上的净利润，我们必须做好哪几件最重要的事情？为什么？

  1. 新产品形成垄断优势，婚恋交友营收冲到30-40亿；
  2. 新品类新商业模式探索成功；
  3. 扩张2个以上新品类：婚庆贡献30-40亿；招聘/服务新品类贡献20-30亿；
  4. 继续强化有缘网既有的优势，渠道多产品覆盖能力和高效率运营体系。
  5. 引入价值观一致的核心业务人员，开展品类扩展

8、这几件事情对我们现有的哪几个部门提出了非常高的要求？主要的实现难度在哪儿？

  1. 新产品部门的产品定位是否足够领先；用户/内容运营能否成为新发动机；以上没有过往经验可参考，必须快速摸索试错
  2. 渠道部门能否配合多产品矩阵形成广泛覆盖能力和ROI深度优化能力；取决于渠道的运营能力能否快速提升
  3. 公司围绕原有产品打造的既有运营体系，如何和新产品的运营体系形成快速对接？ 

请大家认真思考这些问题，并列出题纲（注意：是题纲，由一个一个的词或短句组成。不是发言稿）。我们定于下周一（8月1日）下午4点开始，在泰山会议室开会讨论。

说明：
1、大家在回答这些问题的时候，可能面临信息不全的问题——例如公司其他部门的信息不掌握，以及市场信息不全。请大家尽力提前准备，想一些办法搜集一下相关信息；这也是一个全面了解公司情况和市场情况的机会；
2、大家在回答这些问题的时候，尽量回答到更深层次的本质，而尽量不要涉及战术层面。

绝配到拍拖，青涩到干练，N多的思考、设计、周报、修BUG，无不浸透着你的用心。365天，感谢实现梦想的路上有你同行，一起加油！

组织目标

将团队建设成高战斗力，建立自发展的组织。

通过组织生活会和中层聊天，目前在新业务部门存在两个比较大的问题，也正在着手解决：

组织思考

上周任命杨华为新业务部门（包括商务三部）的政委工作，杨华表态非常乐意接受这个工作，并立马组织了新业务部门产品小同学组织了组织生活会，通过这种沟通形式，还是了解了很多小同学的心声，主要反映在以下几个方面（具体情况可参见杨华周报）：
